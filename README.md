# Engineering Operations Platform

A full-stack Engineering Operations Platform designed for managing teams, projects, tasks, bugs, and incidents within software engineering organizations. The platform provides role-based access control, project tracking, bug management, incident monitoring, and dashboard analytics.

---

## 🚀 Live Demo

### Frontend Application
https://engineering-ops-platform.vercel.app/login

### Backend API
https://engineering-ops-backend.onrender.com

---

## 📌 Features

- JWT Authentication
- Secure Login & Signup
- Role-Based Access Control (Admin, Manager, Developer, QA)
- Team Management
- Project Management
- Task Tracking
- Bug Tracking
- Incident Management
- Dashboard Analytics
- User Profile Management
- Responsive User Interface
- Protected Routes
- REST API Architecture

## 🗄 Database

The application uses PostgreSQL hosted on Neon Database.

Main Entities:

- Users
- Teams
- Projects
- Tasks
- Bugs
- Incidents
- Notifications
- Comments

Prisma ORM is used for schema management and database operations.
